MS Fast Copy — 安装说明
========================

一、安装（重要）
----------------
1. 解压 MSFastCopy.zip（应只有 app + 本说明文件）
2. 将 MSFastCopy.app 拖到「应用程序」文件夹
3. 从「应用程序」中右键 MSFastCopy.app → 打开 → 再点打开
4. 若被拦截：系统设置 → 隐私与安全性 → 底部点「仍要打开」

⚠️ 不要从「下载」或「桌面」直接双击运行，否则菜单栏图标可能不显示。

二、如何确认已在运行
--------------------
- 菜单栏右上角找 **MS** 文字
- 或看 **Dock 栏**是否有 MS Fast Copy 图标
- 终端执行 `pgrep -fl MSFastCopy` 有输出即表示在运行

若菜单栏看不到 MS：
  系统设置 → 控制中心 → 菜单栏微调 → 确认 MSFastCopy 未被隐藏

三、可选：辅助功能权限
----------------------
系统设置 → 隐私与安全性 → 辅助功能 → 勾选 MSFastCopy

四、使用
--------
从 Word / PowerPoint 复制文字，在 Cursor 等应用中 ⌘V 粘贴。

五、退出
--------
Dock 栏右键图标 → 退出；或菜单栏 MS → 退出。

若提示已在运行但找不到图标，先执行：
  pkill MSFastCopy
然后重新从「应用程序」打开。
